### 클라이언트 질문 답변

**Q: 프로젝트와 관련된 사업 분야 업무 경험에 대해 설명해주세요.**

SaaS 기반 B2B 플랫폼 구축 및 운영 경험이 다수 있으며, 본 프로젝트의 핵심 요소인 멀티테넌트 아키텍처, 복잡한 비즈니스 로직, 외부 API 연동 분야에서 직접적으로 관련된 경험을 보유하고 있습니다.

1) 멀티테넌트 SaaS 플랫폼 — Harmony Link (시니어 주간보호 관리 플랫폼)
복수 시설(테넌트)을 하나의 시스템에서 통합 관리하는 멀티테넌트 SaaS 플랫폼을 설계·구현했습니다. 133,000+ LOC, 140+ API 엔드포인트 규모이며, 관리자 웹과 사용자 앱의 이원 구조가 본 프로젝트의 플랫폼 관리자/입점사/고객 구조와 동일합니다. 실시간 알림(FCM) 시스템도 구현하여 LINE/이메일 알림에 동일 패턴을 적용할 수 있습니다.

2) 엔터프라이즈 B2B SaaS — EZ-Approve (전자결재 플랫폼)
8종 결재 행위 워크플로우, 7종 역할별 권한 관리(RBAC), 50+ 페이지 관리자 대시보드를 9주 만에 구현했습니다(504 PR, 120-150 API). 다국어 지원(한/영) 경험도 보유하여 한국어/일본어 다국어 체계에 동일 적용 가능합니다.

3) 규제 준수 + 복잡한 비즈니스 로직 — Series-B (VC 펀드 관리 플랫폼)
VICS 규제 보고 5종 양식 자동화, 펀드 라이프사이클 관리(출자금·투자·성과·정산), NICE KYC 연동 등 복잡한 규제 준수 비즈니스 로직을 14개월간 구현했습니다(1,652 PR, 80+ 엔티티). 일본 통관 규정(합산 과세, 소비세) 대응 로직 설계에 이 경험이 직접 활용됩니다.
